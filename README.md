# encore_bigdata2
## Install
```Bash
$ pip install hello-pysatellite
```
## Dev.
```Bash
$ git clone ...
$ cd encore_bigdata2
$ pdm venv create
$ source .venv/bin/activate
(encore_bigdata-3.8) $ pdm install
```

## Test
```Bash
$ pdm add -dG test pytest pytest-cov
$ pytest
$ pytest -s
$ pytest --cov
```

## Deploy
```Bash
$ pdm publish
```
